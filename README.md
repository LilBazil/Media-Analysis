# AI Creative Portfolio Analyst

Веб-приложение на Flask для анализа творческого портфолио.

Пользователь загружает CSV или XLSX файл с данными о творческих проектах или использует предлагаемые для примера демо-датасеты, задаёт инструкцию или выбирает из шаблонов, после чего LLM-агент проводит анализ и выдаёт текстовый отчёт с анализом и рекомендациями с возможностью вывода графиков

## Что делает проект

- Загружает CSV/XLSX датасет
- Читает пользовательскую инструкцию
- Подключает LLM через OpenRouter API
- Анализирует полный датасет
- Строит графики через matplotlib
- Показывает итоговый отчёт и журнал действий

## Пример задачи

Пользователь может написать:

```text
Какие проекты лучше оставить в портфолио для поиска работы UI-дизайнером? Найди недооценённые работы и стили, которые дают больше вовлечённости.
```

## Рекомендуемые колонки датасета

```text
project_name, category, style, software, views, likes, comments, saves, date, description
```

Если колонки отличаются, агент попытается адаптировать анализ под доступную структуру данных.

## Установка

Создайте виртуальное окружение:

```bash
python -m venv venv
```

Активируйте его на Windows:

```bash
venv\Scripts\activate
```

Установите зависимости:

```bash
pip install -r requirements.txt
```

## Настройка API

1. Получите API-ключ OpenRouter
2. Скопируйте файл `.env.example` в `.env`
3. Вставьте ключ:

```env
OPENROUTER_API_KEY=здесь_писать_ключ
OPENROUTER_MODEL=minimax/minimax-m2.5:free
```

Модель можно заменить на другую доступную в OpenRouter

## Запуск

```bash
python app.py
```

Откройте в браузере:

```text
http://127.0.0.1:5000
```

## Как проверить без своего файла

На главной странице отметьте галочку:

```text
Использовать демо-датасет вместо загрузки файла
```

и нажмите «Запустить анализ»
