# AI Creative Portfolio Analyst

Веб-приложение на Flask для агентного анализа творческого портфолио.

Пользователь загружает CSV/XLSX с данными о творческих проектах или использует демо-датасет, задаёт инструкцию, после чего LLM-агент через Python-интерпретатор анализирует полный DataFrame, строит графики и формирует рекомендации.

## Что делает проект

- Загружает CSV/XLSX датасет.
- Показывает предпросмотр данных.
- Принимает пользовательскую инструкцию/контекст анализа.
- Подключает LLM через OpenRouter API.
- Реализует агентный цикл: LLM сама выбирает, когда вызвать Python-инструмент, и может сделать несколько вызовов.
- Анализирует полный DataFrame `df`, а не сокращённый текстовый фрагмент.
- Строит графики через matplotlib.
- Показывает итоговый отчёт и журнал действий агента.
- Содержит базовую защиту от prompt-injection.

## Пример задачи

Пользователь может написать:

```text
Какие проекты лучше оставить в портфолио для поиска работы UI-дизайнером? Найди недооценённые работы и стили, которые дают больше вовлечённости.
```

## Рекомендуемые колонки датасета

```text
project_name, category, style, software, views, likes, comments, saves, date, description
```

Если колонки отличаются, агент попытается адаптировать анализ под доступную структуру данных.

## Установка

Создайте виртуальное окружение:

```bash
python -m venv venv
```

Активируйте его на Windows:

```bash
venv\Scripts\activate
```

Установите зависимости:

```bash
pip install -r requirements.txt
```

## Настройка API

1. Получите API-ключ OpenRouter.
2. Скопируйте файл `.env.example` в `.env`.
3. Вставьте ключ:

```env
OPENROUTER_API_KEY=your_openrouter_api_key_here
OPENROUTER_MODEL=minimax/minimax-m2.5:free
```

Модель можно заменить на другую доступную в OpenRouter.

## Запуск

```bash
python app.py
```

Откройте в браузере:

```text
http://127.0.0.1:5000
```

## Как проверить без своего файла

На главной странице отметьте галочку:

```text
Использовать демо-датасет вместо загрузки файла
```

и нажмите «Запустить агентный анализ».

## Как проект закрывает ТЗ

- Есть интерфейс взаимодействия — Flask-сайт.
- Пользователь может загрузить CSV/XLSX датасет.
- Есть пользовательская инструкция/контекст.
- LLM используется программно через API.
- LLM работает как агент: сама выбирает Python-вызовы и делает цепочку действий.
- Агент имеет доступ к интерпретатору кода.
- Анализ выполняется по полному DataFrame.
- Есть отчёт, графики и журнал действий агента.
- Есть защита от prompt-injection: данные таблицы считаются данными, а не инструкциями; запрещены системные, файловые и сетевые операции в Python-инструменте.

## Важное замечание о безопасности

Проект учебный. В нём есть базовые ограничения Python-инструмента, но перед реальным продакшен-использованием нужно запускать код агента в полноценной песочнице.
